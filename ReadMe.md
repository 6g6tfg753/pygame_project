# Проект Pygame  
  
## Описание:  
### top-down игра с элементами шутера и головоломками 
### стандартное WASD (игрок); стрельба space
### враги перемещаются когда происходит движение игрока
### при пересечении игрока с монеткой ее можно собрать
### при соприкосновении с камнем можно его подвинуть
### существует 5 карт поля, которые меняются по мере прохождения уровней

   
## Используемые модули:  
### Библиотеки Pygame, copy, sys, collide, анимации, спрайты, хранение данных в txt файле, ООП,


## Классы и функции:  
### Класс BOARD:
#### --> основные объекты:
#####   * карта поля (9Х9)
#####   * группы спрайтов:
######       -- все спрайты (для отрисовки)
######       -- утилиты (в верхнем левом углу поля; монетки и убитые враги)
######       -- враги
######       -- стенки
######       -- объекты
######       -- финиш

#### starting -- начальное меню
#### finishing -- подсчет очков, убитых игроков и монет

#### render_init -- создание игрока, обновление поля, задание всех параметров нулевыми значениями
#### render -- обновление экрана, коллизия объектов, движение врагов, пропуск уровней, отмотать на уровень назад

#### can_go -- проверка на то, что игрок может двигаться в конкретном направлении
#### find_object_sprite -- найти конкретный спрайт из группы, с которым нашлось пересечние
#### add_util -- добавить новый утилит в верхний левый угол
#### exists -- проверяет есть ли такой объект на поле, к которому мы хотим обратиться по индексу

### Класс MAPS:
#### --> основные объекты:
#####   * карта поля
#####   * группы спрайтов

#### field_init -- инициализация поля
#### --> 5 уровней:
#####   * скачивание карты из файлов
#####   * создание спрайтов объектов, врагов, камней, кнопок
